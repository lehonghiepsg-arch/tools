🔑 Lấy Facebook Long-lived Access Token

1
Vào Graph API Explorer
Truy cập: developers.facebook.com/tools/explorer → chọn app của bạn
2
Thêm permissions
Tích chọn: ads_management, ads_read, pages_read_engagement
3
Generate → Exchange lấy token 60 ngày
Click "Generate Access Token" → dùng Access Token Debugger để đổi sang Long-lived token (60 ngày)
4
Lưu vào n8n Credentials
n8n → Credentials → New → Header Auth → Name: fb-token, Value: Bearer YOUR_TOKEN
⚠️
Token hết hạn sau 60 ngày!

Tạo thêm 1 workflow n8n nhắc bạn refresh token trước 5 ngày. Hoặc dùng Facebook Business SDK để auto-refresh.

Workflow 1 — Upload Ads
Tự động upload video lên Facebook Ads
ℹ️
Luồng hoạt động:

Bạn điền thông tin vào Google Sheet → n8n phát hiện hàng mới → upload video lên FB → tạo Ad → cập nhật trạng thái vào Sheet.

Sơ đồ nodes n8n
⏰ Schedule (30 phút/lần)
→
📊 Google Sheets: đọc hàng "pending"
→
🔀 IF: có video mới?
YES →
🌐 HTTP: Upload video lên FB
→
⚙️ Code: lấy video_id
→
🌐 HTTP: Tạo Ad Creative
→
🌐 HTTP: Tạo AdSet
→
🌐 HTTP: Tạo Ad
→
📊 Sheets: cập nhật "done" + Ad ID
1
Node: Schedule Trigger
Chạy mỗi 30 phút để kiểm tra Google Sheet có video mới không.
Cài đặt Schedule Trigger
Trigger Mode: Interval Interval: 30 minutes # Hoặc dùng Cron: */30 * * * *
2
Node: Google Sheets — Đọc hàng "pending"
Lọc các video chưa upload (cột Status = "pending").
Google Sheets node config
Operation: Get Many Rows Spreadsheet ID: YOUR_SPREADSHEET_ID Sheet Name: Upload Queue Filters: Column: Status Value: pending Return All: false Limit: 5 ← upload tối đa 5 video/lần
3
Node: HTTP Request — Upload Video lên Facebook
Upload file video từ URL (Google Drive / Dropbox) lên Facebook Marketing API.
HTTP Request — Upload Video
Method: POST URL: https://graph-video.facebook.com/v18.0/YOUR_AD_ACCOUNT/advideos Body (Form Data): file_url: {{ $json["Video_URL"] }} ← URL video từ Sheet title: {{ $json["Ten_Video"] }} access_token:YOUR_ACCESS_TOKEN # Video_URL phải là link public (Google Drive: share → Anyone with link) # Định dạng: MP4, tối đa 4GB, 16:9 hoặc 9:16
💡
Lưu video trên Google Drive:

Upload video CapCut lên Drive → Share → "Anyone with link" → Copy link → dán vào Sheet cột Video_URL. n8n sẽ tự fetch và upload lên FB.

4
Node: HTTP Request — Tạo Ad Creative
Sau khi upload video thành công, dùng video_id để tạo creative.
HTTP Request — Ad Creative
Method: POST URL: https://graph.facebook.com/v18.0/act_YOUR_AD_ACCOUNT/adcreatives Body (JSON): { "name": "{{ $json["Ten_Video"] }}_creative", "object_story_spec": { "page_id": "YOUR_PAGE_ID", "video_data": { "video_id": "{{ $json["id"] }}", ← từ bước 3 "message": "{{ $json["Caption"] }}", ← từ Sheet "call_to_action": { "type": "SEND_MESSAGE" } } }, "access_token": "YOUR_ACCESS_TOKEN" }
5
Node: HTTP Request — Tạo Ad (ghép AdSet + Creative)
Tạo Ad thực sự trong campaign đã có sẵn của bạn.
HTTP Request — Create Ad
Method: POST URL: https://graph.facebook.com/v18.0/act_YOUR_AD_ACCOUNT/ads Body (JSON): { "name": "{{ $json["Ten_Video"] }}_{{ new Date().toLocaleDateString('vi') }}", "adset_id": "{{ $json["AdSet_ID"] }}", ← từ Sheet "creative": { "creative_id": "{{ $json["id"] }}" }, "status": "PAUSED", ← tạm dừng, bạn bật thủ công "access_token":"YOUR_ACCESS_TOKEN" }
6
Node: Google Sheets — Cập nhật trạng thái
Ghi lại Ad ID và đổi Status thành "done" để không upload lại.
Google Sheets — Update Row
Operation: Update Row Match Column: Ten_Video ← tìm đúng hàng Update: Status: done Ad_ID: {{ $json["id"] }} Upload_Time: {{ new Date().toLocaleString('vi') }}
Workflow 2 — Báo cáo ROAS
Báo cáo hàng ngày tự động
Sơ đồ nodes
⏰ Schedule 7:30 sáng
→
🌐 HTTP: lấy data FB Ads
→
⚙️ Code: tính ROAS
→
🤖 Claude: viết báo cáo
→
📊 Sheets: lưu số liệu
→
💬 Messenger/Zalo: gửi báo cáo
1
Node: Schedule — 7:30 sáng mỗi ngày
Cron expression
Mode: Custom (Cron) Expression: 30 7 * * * ← 7:30 sáng mỗi ngày
2
Node: HTTP Request — Lấy insights từ Facebook
Gọi Marketing API để lấy chi phí, click, impression của ngày hôm qua.
FB Ads Insights API
Method: GET URL: https://graph.facebook.com/v18.0/act_YOUR_AD_ACCOUNT/insights Query Params: fields: campaign_name,spend,clicks,impressions,ctr,cpc,actions date_preset: yesterday level: campaign access_token:YOUR_ACCESS_TOKEN # "actions" chứa số đơn/message — dùng để tính ROAS
3
Node: Code — Xử lý & tính ROAS
JavaScript xử lý data từ FB API, tính toán và format cho Claude.
Code Node — JavaScript
// Xử lý data từ Facebook Insights const campaigns = $input.all(); const results = campaigns.map(item => { const d = item.json; // Lấy số messages/purchases từ actions const messages = d.actions?.find( a => a.action_type === 'onsite_conversion.messaging_conversation_started_7d' )?.value || 0; return { campaign: d.campaign_name, spend: parseFloat(d.spend || 0), clicks: parseInt(d.clicks || 0), ctr: parseFloat(d.ctr || 0).toFixed(2) + '%', cpc: parseFloat(d.cpc || 0).toFixed(0) + 'đ', messages: parseInt(messages), // ROAS ước tính: (messages × AOV) / spend // Thay AOV = giá trị đơn hàng trung bình của bạn (VD: 300000) roas_est: messages > 0 ? ((messages * 300000) / parseFloat(d.spend)).toFixed(1) + 'x' : 'N/A' }; }); return [{ json: { campaigns: results, date: new Date().toLocaleDateString('vi') } }];
4
Node: HTTP Request — Claude viết báo cáo
Gửi data lên Claude API, nhận về báo cáo tiếng Việt dễ đọc.
Claude API Call
Method: POST URL: https://api.anthropic.com/v1/messages Headers: x-api-key: YOUR_ANTHROPIC_API_KEY anthropic-version: 2023-06-01 content-type: application/json Body (JSON): { "model": "claude-haiku-4-5-20251001", ← rẻ + nhanh "max_tokens": 500, "messages": [{ "role": "user", "content": "Ngày {{ $json.date }}, đây là kết quả Facebook Ads:\n{{ JSON.stringify($json.campaigns) }}\n\nViết báo cáo ngắn tiếng Việt: tổng chi phí, chiến dịch tốt nhất, chiến dịch cần xem lại, và 2 đề xuất cụ thể. Dùng emoji cho dễ đọc." }] }
5
Node: Google Sheets — Lưu số liệu hàng ngày
Append Row vào Ads Report sheet
Operation: Append Row Sheet: Ads Report Values: Date: {{ $json.date }} Total_Spend: {{ $json.campaigns.reduce((s,c) => s+c.spend, 0) }} Best_Campaign: {{ $json.campaigns[0].campaign }} Report: {{ $('Claude').item.json.content[0].text }}

Xử lý lỗi thường gặp
Lỗi phổ biến và cách fix
!
Lỗi: "Invalid OAuth access token"
Token hết hạn hoặc sai. Vào Graph API Explorer → tạo token mới → cập nhật vào n8n Credentials.
Kiểm tra token còn hạn không
# Dán vào browser để kiểm tra: https://graph.facebook.com/debug_token?input_token=YOUR_TOKEN&access_token=YOUR_TOKEN # Xem field "expires_at" — nếu trong quá khứ: token hết hạn
!
Lỗi: Video upload thất bại (timeout)
Video quá lớn hoặc link Drive chưa public. Kiểm tra 2 điều:
  • File video phải < 4GB, định dạng MP4 (H.264)
  • Google Drive: click "Share" → "Anyone with the link" → chọn "Viewer"
  • Dùng link dạng: https://drive.google.com/uc?export=download&id=FILE_ID
!
Lỗi: "Ad account must be on an allowlist"
Tài khoản Ads chưa được verify hoặc bị giới hạn. Cần vào Business Manager → xác minh danh tính → đợi Facebook duyệt.
!
Lỗi: Claude API "authentication_error"
Sai API key hoặc chưa có. Lấy key tại console.anthropic.com → API Keys → Create Key.
⚠️
Chi phí Claude API:

Dùng model claude-haiku-4-5-20251001 — rẻ nhất, đủ để viết báo cáo. Mỗi báo cáo khoảng $0.001–0.003 (~25–75 đồng). 1 tháng < 5.000đ.
